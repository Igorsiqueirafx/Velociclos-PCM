Manual do Usuário

Guia completo para instalar, configurar e operar o Expert Advisor Velociclos PCM no MetaTrader 5.

v7.33 Versão Atual
MT5 Plataforma
5 Configurações

Instalação

Siga os passos abaixo para instalar o E.A. no seu MetaTrader 5.

1

Baixe o arquivo

Clique no botão de download na página do Expert Advisor para baixar o arquivo Velociclos PCM.ex5.

2

Abra a pasta do MetaTrader

No MetaTrader 5, vá em Arquivo → Abrir Pasta de Dados → MQL5 → Experts.

Atalho: pressione Ctrl+Shift+D no MT5 para abrir a pasta rapidamente.
3

Cole o arquivo

Copie o arquivo Velociclos PCM.ex5 para a pasta Experts que abriu.

4

Reinicie o MetaTrader

Feche e abra novamente o MetaTrader 5. O E.A. aparecerá na janela Navegador → Expert Advisors.

5

Arraste para o gráfico

Arraste o E.A. para o gráfico desejado (recomendado: XAUUSD M5). Ative o Trading Automático na barra de ferramentas.

Antes de operar em conta real, teste sempre em conta demo.

Modos de Operação

Escolha o modo que melhor se adapta ao seu estilo de trading.

Automático

O E.A. identifica ciclos automaticamente e abre ordens com base nos últimos candles do dia. Ideal para quem quer operar com menos intervenção manual.

  • Detecção automática de ciclos
  • Entradas baseadas em padrões
  • Menor necessidade de monitoramento

Arrastar Contínuo

Você escolhe o início do canal arrastando uma linha vertical no gráfico. O sistema forma o canal e aguarda as entradas com base na sua marcação.

  • Controle total sobre o canal
  • Marcação manual flexível
  • Ideal para traders experientes

Configurações

Parâmetros recomendados para operar com segurança.

Gestão de Risco

O E.A. oferece dois métodos de gestão de risco:

Risco Percentual Define o lote automaticamente com base no % de risco por operação.
Lote Fixo Utiliza sempre o mesmo volume, independente do stop.
Gestão de Stop
Máximo de Stops / Dia 3 stops
Limite Semanal Bloqueio automático ao atingir o limite

Quando o limite é atingido, o E.A. bloqueia novas operações até o próximo período.

Break Even

O recurso Break Even move o stop loss para o ponto de entrada (zero risco) após a operação avançar o suficiente.

Ativação Automática após atingir o lucro mínimo configurado
Modo Minimalista

Quando ativado, o painel de informações do E.A. mostra apenas os dados essenciais no gráfico, facilitando a visualização.

Painel Apenas lucro, perda e status da operação

Dicas Importantes
